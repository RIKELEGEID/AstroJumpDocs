Introduction

Purpose of the Documentation

This documentation provides a detailed technical description of the game developed using Pygame. It is intended for the committee that's going to be grading my graduation final project from software development.

Overview of the Game

"AstroJump" is an action, pixel art, 2D platformer game where players control a character navigating through various levels, avoiding obstacles and enemies. The game is targeted towards players who enjoy challenging platformers. Key features include dynamic level transitions, interactive menus, sound effects and background music.
