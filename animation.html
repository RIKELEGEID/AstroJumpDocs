animation module

class animation.Animation(images, img_dur=5, loop=True)[source]

Bases: object

Manages an animation sequence from a list of images, with control over the frame duration and looping behavior.

Parameters:

  • images – A list of Pygame Surface objects that make up the animation frames.

  • img_dur – The duration of each frame in the animation in game frames, defaults to 5.

  • loop – A boolean that determines whether the animation should loop or play once, defaults to True.

copy()[source]

Creates a copy of this Animation object, allowing for independent animation sequences using the same frames.

Returns:

A new instance of Animation with the same images, duration, and looping behavior.

img()[source]

Retrieves the current image of the animation based on the current frame index.

Returns:

The Pygame Surface object of the current frame.

update()[source]

Updates the animation frame, advancing it or looping it depending on the configuration.

path – The path to the directory containing image files, relative to the base image directory.

Returns:
