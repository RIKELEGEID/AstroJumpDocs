player module

class player.Player(x, y, width, height, screen)[source]

Bases: object

Represents a player character in a game with capabilities such as movement, jumping, and animation handling based on the state.

Parameters:

  • x – Initial x-coordinate of the player.

  • y – Initial y-coordinate of the player.

  • width – Width of the player.

  • height – Height of the player.

  • screen – Pygame screen where the player is to be drawn.

calculate_new_position()[source]

Calculates the new position of the player based on current movement and gravity.

Returns:

Tuple (new_x, new_y) representing the new position coordinates.

draw(camera)[source]

Draws the player on the screen based on the current animation frame and camera view.

Parameters:

camera – Camera object that handles viewport and projection transformations.

handle_event(event)[source]

Handles player input events to update movement and jumping status.

Parameters:

event – Pygame event to be processed.

is_moving()[source]

Checks if the player is currently moving or jumping.

Returns:

True if the player is moving or jumping, False otherwise.

reset_position()[source]

Resets the player’s position to the initial coordinates.

update_animation()[source]

Updates the player’s animation state based on the movement and jumping flags.

update_position(new_x, new_y)[source]

Updates the player’s position on the screen.

Parameters:

  • new_x – New x-coordinate.

  • new_y – New y-coordinate.
