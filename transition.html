transition module

transition.circular_fade(screen, mode='in', duration=500, max_radius=None, border_color=(255, 255, 255), border_thickness=15)[source]

Performs a circular fade effect on a Pygame screen object. The fade can be configured to fade in, fade out, or both sequentially. This function is typically used for transition effects between scenes or menus.

Parameters:

  • screen – The Pygame surface where the fade effect is applied.

  • mode – A string indicating the fade mode; “in” for fading in, “out” for fading out, “both” for both in sequence.

  • duration – The duration of the fade effect in milliseconds.

  • max_radius – The maximum radius of the circle used in the fade effect. If None, it calculates based on screen dimensions.

  • border_color – The color of the border of the fading circle.

  • border_thickness – The thickness of the circle’s border during the fade effect.
