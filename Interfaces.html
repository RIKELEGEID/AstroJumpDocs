Interfaces

User Interface

The player interacts with the game through a graphical interface with buttons.

There are multiple buttons like the "Settings" button that opens the settings menu in which the player can change the music and SFX volume via sliders.

The "Tutorial" button which shows the player a very simple description of the games controls.

The "Level select" button which gives the player the option to chose from 3 different levels.

And lastly the "Quit" button which exits the game.

Programmer Interfaces (API)

The game leverages Pygame's built-in functionalities as well as custom modules for specific tasks like handling physics or managing game states.
