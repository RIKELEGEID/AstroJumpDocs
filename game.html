show_win_menu()[source]

Displays the win menu after a game victory. This menu provides options to return to the main menu or quit the game.

The method continuously checks for mouse position and button clicks to respond to user interactions: - Displays a “YOU WON!!!” message and two buttons: “Menu” and “Quit”. - If the “Menu” button is clicked, it resets the player’s position, plays a button sound, starts background music, applies a fade-out effect, and returns to the main menu. - If the “Quit” button is clicked, it quits the application.

This loop runs indefinitely until an exit condition is triggered, (either button click or quitting the event).
